===================
Pilot Documentation
===================

The DIRAC (Distributed Infrastructure with Remote Agent Control) Pilot is what creates the possibility
for a uniform view of computing resources. The DIRAC Workload Management system realizes the task scheduling paradigm with Generic Pilots.
